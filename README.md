## Installation 
1. Clone the project
```bash
 git clone https://github.com/AidaSousa/musicForAGoodDayLaravel.git
```
2. Install dependencies
```
 composer install
```
3. Run the PHP server and keep this terminal open
```
 php artisan serve
```
